# Route and Riches

Route and Riches is a JavaFX transport management game where the player builds roads, places stops, creates routes, buys vehicles, and grows a city transport network over time.

## Features

- Main menu with new game, load game, and exit options
- Tile based city map with roads, buildings, greenery, and stops
- Road and stop placement tools
- Route creation and route management
- Bus and tram vehicle support
- Vehicle movement along assigned routes
- Passenger spawning, pickup, delivery, and income generation
- Minimap navigation
- Save and load support
- Traffic light placement and timing controls
- City growth system based on transport activity
- Auto simulation mode for demonstrating gameplay progression

## Requirements

- Java JDK 21
- JavaFX SDK 21
- Windows PowerShell
